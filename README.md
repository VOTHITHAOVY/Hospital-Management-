##  GIỚI THIỆU DỰ ÁN 

Hệ thống Quản lý Bệnh viện (Hospital Management System) là giải pháp toàn diện nhằm số hóa và tối ưu hóa mọi hoạt động trong bệnh viện, từ quản lý bệnh nhân, khám chữa bệnh đến quản lý thuốc và tài chính.

##  MỤC TIÊU CHIẾN LƯỢC

- **Số hóa toàn diện**: Chuyển đổi từ quản lý thủ công sang hệ thống điện tử
- **Tối ưu hóa quy trình**: Từ khám bệnh đến thanh toán
- **Nâng cao trải nghiệm**: Cho cả bệnh nhân và nhân viên y tế
- **Đảm bảo tuân thủ**: Theo quy định của Bộ Y tế

##  HỆ THỐNG PHÂN QUYỀN

| Vai trò | Chức năng chính |
|---------|-----------------|
| **Admin** | Quản trị hệ thống |

##  CÁC MODULE TRỌNG TÂM

### 3.1 MODULE QUẢN LÝ KHÁM CHỮA BỆNH

**Quy trình khám bệnh tiêu chuẩn:**
Đăng ký → 2. Tiếp nhận → 3. Khám lâm sàng → 4. Xét nghiệm → 5. Chẩn đoán → 6. Kê đơn → 7. Thanh toán → 8. Kết thúc

**Tính năng nổi bật:**
-  Đặt lịch hẹn thông minh
-  Quản lý hàng đợi thời gian thực
-  Hồ sơ bệnh án điện tử (EMR)
-  Kết nối phòng xét nghiệm

### 3.2 MODULE QUẢN LÝ DƯỢC PHẨM

**Hệ thống cảnh báo thông minh:**
-  Thuốc sắp hết hạn
-  Tồn kho thấp
-  Tương tác thuốc
-  Chống kê đơn trùng lặp

**Quy trình cấp phát thuốc:**

1.Bác sĩ kê đơn → 2. Kiểm tra tồn kho → 3. Bệnh nhân thanh toán → 4. Dược sĩ chuẩn bị → 5. Phát thuốc

### 3.3 MODULE XÉT NGHIỆM & CẬN LÂM SÀNG

**Quy trình xét nghiệm:**
1.Chỉ định → 2.Tiếp nhận → 3.Thực hiện → 4.Nhập kết quả → 5.Duyệt → 6.Trả kết quả


**Tính năng đặc biệt:**
-  Integration với máy xét nghiệm
-  Cảnh báo kết quả bất thường
-  Tự động thông báo khi có kết quả
-  Lưu trữ đa phương tiện (PDF, hình ảnh)

##  TÍCH HỢP HỆ THỐNG & CÔNG NGHỆ

### 4.1 Tích hợp bên ngoài

**Cổng thanh toán:**
-  Ngân hàng: ATM, Internet Banking
-  Ví điện tử: Momo, ZaloPay, VNPAY
-  POS thanh toán thẻ

**Kết nối hệ thống:**
-  Bảo hiểm y tế (BHYT)
-  Cổng dữ liệu Bộ Y tế
-  Lab analyzer machines

### 4.2 Hệ thống thông báo đa kênh

- **SMS**: Nhắc lịch hẹn
- **Email**: Kết quả xét nghiệm  
- **Push Notification**: Trên mobile app
